# v1.0.0

Release date: 2026-06-14

## Summary

This is the first stable release of the Terraform module library.

It establishes the versioning model and the consumer pattern for product teams.

## Included Modules

This release exposes the full module catalog already present in the repository,
with optional toggles controlling which resources are created.

## Consumer Baseline

The initial demo consumer uses:

- `create_vpc = true`
- `create_ec2 = true`
- all other `create_*` flags set to `false`

## Breaking Changes

None. This is the first stable release.

## Upgrade Notes

- Pin consumers to `ref=v1.0.0`
- Change release versions intentionally after validation
- Keep environment-specific values in tfvars files
